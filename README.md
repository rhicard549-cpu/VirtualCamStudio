\# VirtualCam Studio



A Windows desktop application for previewing, processing, and outputting images and videos to a virtual camera.



\## Features



\- Drag \& Drop Images

\- Drag \& Drop Videos

\- Smart Auto Fit

\- Portrait Canvas

\- Face Centering

\- Virtual Camera Output

\- Virtual Microphone

\- ADB Integration

\- Cloud Phone Testing
